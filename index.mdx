## 技术栈

- **区块链**: BNB Smart Chain (BSC)
- **智能合约**: Solidity ^0.8.17 + OpenZeppelin
- **前端**: Next.js 14 + Wagmi + Viem
- **后端**: Node.js v22 + OpenClaw (Agent Runtime)
- **协议/标准**: ERC-20 (Token), ERC-8004 (Agent Identity)

## 功能

- **持有即用**：持有价值 10 美元的 HODLAI 代币，即可获得每日约 1 美元的 AI 计算额度。无需每月支付费用。
- **税金入库**: 3% 的交易税将自动注入 API 金库（`...Ce8F0`），用于支付上游提供商。
- **钻石手机制**: API 配额将逐步解锁（24 小时内从 0% 解锁至 100%），以激励长期持有。
- **去中心化代理身份**：符合 BNB 链上的 **ERC-8004** 标准，可实现无需信任的代理验证。
- **Gas 高效**：针对 BNB 智能链的低延迟、高吞吐量环境进行了优化。
